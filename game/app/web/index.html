玩法说明

🎯 目标

  • 消灭敌方所有单位
  • 或者占领所有城堡（站在城堡上一回合）

🗺️ 地形

  • 平地（浅绿）：移动消耗 1，无加成
  • 森林（深绿）：移动消耗 2，防御 +2
  • 山地（灰）：移动消耗 3，防御 +3
  • 河流（蓝）：移动消耗 3，无加成
  • 城堡（金）：防御 +5，不可被敌方占领后踏入

⚔️ 单位

  • 剑士 ×2：攻防均衡，近战，移动 3
  • 弓箭手 ×1：远程（射程 2），低防，移动 3，技能【狙击】（+1 射程）
  • 骑士 ×1：高移速（5），高攻，低防，技能【连击】（攻击 2 次，每次 50% 伤害）
  • 治疗师 ×1：可治疗相邻友军 20HP，技能【集结】（相邻友军 +10% 攻击）

📐 回合流程

  1. 服务器生成地图和单位
  2. 当前玩家可无限次移动/攻击/技能/待命
  3. 点「结束回合」→ 下一玩家
  4. 所有玩家结束 → 服务器结算（升级、清除阵亡）→ 下一回合

💥 战斗公式

伤害 = ATK × (ATK / (ATK + DEF + 地形)) × 兵种克制 × 暴击

  • 克制：剑士→骑士(+20%)，骑士→弓箭手(+20%)
  • 暴击：基础 5% + 每级 1%，伤害 1.5 倍
